# fckgit

A Git client so powerful, it fucks up your workflow management just by existing.

## Installation

```bash
npm install -g fckgit
```

Or if you're feeling adventurous:

```bash
git clone https://github.com/earlyprototype/fckgit.git
cd fckgit
npm install
```

## Usage

```bash
fckgit init
fckgit commit
fckgit push
```
