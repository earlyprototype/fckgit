================================================================================
                    --faang FLAG ADDED TO STANDALONE TOOL
================================================================================

Silicon Valley mode is now available in the standalone fckgit tool with the 
--faang flag. No MCP server needed!

================================================================================
                              USAGE
================================================================================

Watch Mode (Professional commits on every change):
  python -m fckgit --faang

Single Commit (One professional commit):
  python -m fckgit --once --faang

Regular Mode (for comparison):
  python -m fckgit
  python -m fckgit --once

================================================================================
                           WHAT IT DOES
================================================================================

Transforms your commits from embarrassing to enterprise-grade:

WITHOUT --faang:
  "fix: stuff works now"

WITH --faang:
  fix: Resolve critical system stability issues
  
  Addressed underlying architectural concerns that were impacting system
  reliability. This fix improves fault tolerance and ensures consistent
  behavior under edge-case scenarios, enhancing overall platform stability.

================================================================================
                          CODE CHANGES
================================================================================

Modified Files:
  - fckgit.py:
    * Added generate_silicon_valley_message() function
    * Updated generate_message() to accept faang_mode parameter
    * Updated GitChangeHandler to accept and use faang_mode
    * Updated watch_mode() to accept faang_mode parameter
    * Added --faang flag to argument parser
    * Shows "SILICON VALLEY MODE" indicators when active

Documentation:
  - README.md - Added --faang usage examples
  - SILICON_VALLEY.md - Updated with standalone usage
  - CHANGELOG.md - Documented v0.2.0 changes
  - FAANG_FLAG_USAGE.md - Complete usage guide (NEW)

================================================================================
                         TECHNICAL DETAILS
================================================================================

Function Signatures Changed:
  - generate_message(diff: str, faang_mode: bool = False)
  - watch_mode(faang_mode: bool = False)
  - GitChangeHandler.__init__(faang_mode: bool = False)

New Function:
  - generate_silicon_valley_message(diff: str) -> str
    Uses same Gemini 2.5 Flash-Lite model
    Generates multi-line commits with technical rationale
    Includes enterprise buzzwords

AI Prompt Engineering:
  Professional mode uses enhanced prompt with:
    - "You are a senior engineer at a FAANG company"
    - Enterprise buzzword requirements
    - Technical rationale body paragraph
    - "Make even trivial changes sound important"

================================================================================
                            ALIASES
================================================================================

You can create an alias for easier use:

Bash/Zsh:
  echo 'alias fckgit-pro="python -m fckgit --faang"' >> ~/.bashrc
  source ~/.bashrc
  
  # Then just run:
  fckgit-pro

PowerShell:
  function fckgit-pro { python -m fckgit --faang }
  
  # Then just run:
  fckgit-pro
