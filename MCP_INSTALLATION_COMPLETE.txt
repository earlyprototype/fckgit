THE NUCLEAR OPTION:

1. fckgit_blastoff
   START FULL AUTOMATIC MODE - Launch fckgit watch mode (auto-commits + auto-pushes)

SILICON VALLEY MODE:

2. fckgit_silicon_valley
   Commit with FAANG-tier professional messages (enterprise buzzwords included)
   
3. fckgit_professionalize
   Transform casual messages into impressive enterprise-grade commits

WATCH MODE MANAGEMENT:
   
4. fckgit_watch_status
   Check if fckgit watch mode is currently running
   
5. fckgit_stop_watch
   Stop the running fckgit watch process

MANUAL OPERATIONS:

6. fckgit_status
   Get current git status and optionally the full diff
   
7. fckgit_generate_message
   Generate AI commit message without committing
   
8. fckgit_commit
   Auto-commit with AI-generated message (optional auto-push)
   
9. fckgit_commit_with_message
   Commit with a specific message (no AI)
   
10. fckgit_push
    Push to remote (handles conflicts with auto-rebase)
   
11. fckgit_cleanup_lock
    Clean up stale git lock files
